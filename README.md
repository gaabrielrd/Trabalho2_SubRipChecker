Trabalho2_SubRipChecker
=======================


Trabalho acadêmico desenvolvido para avaliação na disciplina de Programação Orientada a Objetos II.
Tem como objetivo selecionar um arquivo .srt por meio de um jFileChooser e procurar por erros de sintaxe.

Developed to evaluate progress in Object Oriented Programming II, Computer Science, FURB/SC. 
Aims to select a .srt file via a JFileChooser and look for syntax errors.
